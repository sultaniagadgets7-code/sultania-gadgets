# OG Image Instructions

## Quick Setup (2 minutes)

### Option 1: Use the Image from Chat
1. Scroll up in this chat to find the OG image I showed you
2. Right-click on the image
3. Save as: og-image.jpg
4. Move to: sultania-gadgets/public/og-image.jpg
5. Run: vercel --prod

### Option 2: Create Your Own (5 minutes)
1. Go to: https://www.canva.com/
2. Search for "Open Graph Image" or create custom size: 1200x630px
3. Design with:
   - Your logo (use logo.svg from public folder)
   - Text: "Premium Gadgets. Fast Delivery."
   - Text: "Cash on Delivery Available"
   - Background: Dark/Black
   - Add product images if you have them
4. Download as JPG
5. Save as: og-image.jpg
6. Move to: sultania-gadgets/public/og-image.jpg

### Option 3: Use a Simple Template
1. Go to: https://ogimage.gallery/
2. Choose a template
3. Customize with your brand
4. Download
5. Save as: og-image.jpg

### Option 4: Quick Placeholder
For now, you can use your logo as OG image:
1. Copy sultania-gadgets/public/icon1.png
2. Rename copy to: og-image.jpg
3. This works temporarily until you create a proper one
